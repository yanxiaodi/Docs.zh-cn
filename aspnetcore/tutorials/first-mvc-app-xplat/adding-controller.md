# <a name="adding-a-controller-to-an-aspnet-core-mvc-app-with-visual-studio-code"></a>使用 Visual Studio Code 将控制器添加到 ASP.NET Core MVC 应用

作者：[Rick Anderson](https://twitter.com/RickAndMSFT)

[!INCLUDE[adding-controller](../../includes/mvc-intro/adding-controller1.md)]

* 在“VS Code”中，选择“EXPLORER”图标，然后在按住 Control 的同时单击（右键单击）“控制器”，选择“新建文件”，然后将新文件命名为 HelloWorldController.cs。
