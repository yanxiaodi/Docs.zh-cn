<a name="how-do-i-use-the-aspnet-ajax-nobot-control"></a>[如何:]使用 ASP.NET AJAX NoBot 控件？
====================
通过[Joe stagner 将](https://github.com/JoeStagner)

从 ASP.NET AJAX 控件工具包 NoBot 控制可帮助防止对你的 Web 应用程序的某些类型的未经授权的访问。 在本视频中我们看到如何 NoBot 控件提供了多种内置防御自动访问，包括检查的数量和频率的窗体提交。 我们还看到控件如何允许开发人员定义自定义的挑战，例如 JavaScript 测试，仅当在浏览器中查看网页时，才会成功。
