## <a name="overview"></a>概述

ASP.NET AJAX 控件工具包中的动画控件不只是一个控件，但一个整个框架，以向控件添加动画。 通常情况下，启动动画由触发与同一个控件中的用户交互。 但是也可能与一个控件，然后动画交互另一个控件。
