## <a name="overview"></a>概述

如果可以选择仅一组的选项之一，通常将使用单选按钮。 但没有一个缺点： 选择一个单选按钮组中的后，不能取消选中所有单选按钮。 复选框在任何时候可以是未选中状态，但是不互相排斥。 本教程提供这两种方法的最佳： 是互斥的复选框。
